The 4km ERA5 target grid file (`downscaling/default_target_grid_files/era5_4km_default_target_grid.nc`) is the "legacy" target grid from `/beegfs/CMIP6/arctic-cmip6/era5/` directory. Exact provenance is unknown, but this grid has been used to create the production versions of all 4km downscaled products as of April 2026. This grid is used for regridding when `downscaling/prep_era5_variables.py` is run in legacy mode.

The 12km ERA target grid file (`downscaling/default_target_grid_files/era5_12km_default_target_grid.nc`) was created from the mean temperature output of the wrf-downscaled-era5-curation repo, by using the `downscaling/prep_era5_variables.py` and `downscaling/make_final_target_grid_file.py` scripts. The file committed to this repo was created from `/beegfs/CMIP6/jdpaul3/wrf_era5_4km_daily/t2_mean/t2_mean_1965_daily_era5_4km_3338.nc` in April 2026.
